# OCR 文字识别应用

这是一个基于浏览器的 OCR 文字识别应用，可以从图片中提取文字。

## 功能特点

- 支持多种语言：
  - 简体中文
  - 繁体中文
  - 英文
  - 日文
  - 韩文
- 拖拽上传图片
- 实时识别进度显示
- 完全在浏览器端运行，保护隐私

## 技术栈

- HTML5
- CSS3
- JavaScript
- Tesseract.js

## 使用方法

1. 访问 [应用页面](https://[你的用户名].github.io/ocr-app)
2. 选择要识别的语言
3. 上传或拖拽图片到指定区域
4. 点击"开始识别"
5. 等待识别完成，查看结果

## 本地运行

直接在浏览器中打开 `index.html` 文件即可使用。
